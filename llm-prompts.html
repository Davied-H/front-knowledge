用大白话说就是...

1

System Prompt（系统提示） = 给 AI 设定的"人设"和"规矩"

比如告诉它"你是一个温柔的老师"或"你只能用中文回答"

2

User Prompt（用户输入） = 你在对话框里打的字

就是你问 ChatGPT / Claude 的那些问题和请求

3

Assistant（助手回复） = AI 给你的答复

AI 会综合考虑"人设"和你的问题，然后给出回答

一次对话是这样发生的

🎬 导演指示 （你看不到这个）

"你是一个友好的美食推荐助手，说话要活泼可爱~"

↓
🙋 你说

"我想吃火锅，有什么推荐吗？"

↓
🤖 AI 回复

"哇！火锅爱好者你好呀~ 我超推荐海底捞的番茄锅底..."

看到了吗？AI 的回复风格（活泼可爱）是由"导演指示"决定的！

🎬

System Prompt = 幕后导演的指示

想象你请了一个演员来帮你表演。
在演出开始前，你会私下告诉他："你要扮演一个热情的销售员"

这个指示观众是看不到的，但演员会照着做。这就是 System Prompt！

System Prompt 可以做什么？

👤 设定 AI 的身份（医生、老师、翻译...）
🎭 设定说话风格（正式、幽默、温柔...）
🚫 设定禁止事项（不讨论某些话题）
📝 设定回复格式（列表、简短、详细...）

生活化的例子

👨‍🍳 例子1：让 AI 变成厨师
你是一个有20年经验的家庭厨师。
用户问你做菜的问题时：
- 用简单易懂的话解释
- 给出具体的用量（几勺、几克）
- 提醒可能的常见错误
- 语气要像邻居阿姨一样亲切
设定了这个之后，不管你问什么做菜问题，AI 都会用"邻居阿姨"的风格回答
🌍 例子2：让 AI 变成翻译官
你是一个中英翻译专家。
规则很简单：
- 用户发中文 → 你翻译成英文
- 用户发英文 → 你翻译成中文
- 只输出翻译结果，不要解释
有了这个设定，AI 就变成了一个纯粹的翻译工具，不会废话
👶 例子3：让 AI 给小朋友讲故事
你是一个会讲故事的大姐姐。
- 用5岁小朋友能懂的话
- 故事要有趣、有教育意义
- 可以加入一些象声词（砰！哗啦~）
- 故事结尾要有一个小道理

小技巧：System Prompt 写得越详细，AI 的回答就越符合你的期望！

🙋

User Prompt = 你说的话

这个最简单！就是你在 ChatGPT、Claude、文心一言等 AI 工具里 输入框里打的字。

你平时是不是这样问 AI 的？

"帮我写一封请假邮件"
"Python 怎么读取 Excel 文件？"
"给我推荐几部好看的电影"
"这段话帮我翻译成英文"

☝️ 这些都是 User Prompt！

怎样问问题，AI 回答得更好？

❌ 模糊的问法
"写个文案"

AI：？？？ 什么文案？给谁看的？

✅ 清晰的问法
"帮我写一个奶茶店开业的朋友圈文案，要活泼可爱，100字左右"

AI 立刻就知道该怎么写了！

❌ 模糊的问法
"这代码有问题"

AI：什么代码？什么问题？

✅ 清晰的问法
"这段 Python 代码运行时报错 'list index out of range'，帮我看看哪里有问题：[贴上代码]"

提问小技巧

  • • 说清楚背景 - "我是一个刚学编程的大学生..."
  • • 说清楚目的 - "我想用来发朋友圈 / 交作业 / 给客户看"
  • • 说清楚要求 - "字数500左右 / 要正式一点 / 要有例子"
  • • 给个例子 - "类似这种风格：[例子]"
🤖

Assistant = AI 的回复

这个更简单了！就是 AI 说的话。

"今天天气怎么样？"
"抱歉，我无法获取实时天气信息。但你可以查看手机自带的天气应用，或者告诉我你在哪个城市，我可以告诉你这个季节通常的天气情况~"

☝️ 灰色气泡就是 Assistant（AI 的回复）

进阶技巧：用例子教 AI

有时候，与其用文字描述你想要什么，不如直接给 AI 看几个例子。这叫做 "Few-shot Learning"（少样本学习）。

就像教小朋友做题一样：
与其说 "把数字变成中文大写"
不如说 "看例子：1 → 壹，2 → 贰，3 → 叁，那么 4 → ？"

用对话的方式"教" AI：

"今天心情很好！" → 判断情绪
积极
"快递又丢了，烦死了" → 判断情绪
消极
"明天要开会" → 判断情绪
???（AI 会学着回答：中立）

通过前面的例子，AI 学会了你想要的回答格式！

日常使用中，你需要关心 Assistant 吗？

普通用户

不用管！你只要问问题，看 AI 的回答就行了。Assistant 就是 AI 说的话，很自然~

开发者 / 高级用户

可以利用 Assistant 消息来"预设"对话历史，教 AI 按特定方式回答。

日常使用 AI 的小技巧

1

先说清楚你是谁

"我是一个不懂代码的产品经理..."
"我是高中生，在准备英语考试..."

2

告诉 AI 你想要什么样的回答

"用简单的话解释，不要用专业术语"
"给我列一个表格对比"
"分点说明，每点不超过2句话"

3

不满意就追问

"说得更详细一点"
"举个具体的例子"
"换一种说法，更通俗一些"

4

给 AI 一个角色

"假设你是一个有10年经验的健身教练..."
"请用小学老师的语气解释..."

实战演示

一个完整的好问题：

我是一个刚开始健身的女生，体重55kg。

我想在家锻炼，没有器械，每天只有30分钟时间。

请帮我制定一个一周的锻炼计划。

要求：
- 每天的动作不超过5个
- 要说明每个动作做几组、几个
- 用表格形式展示
这个问题包含了： 背景信息 + 具体需求 + 格式要求 ，AI 一定能给出满意的回答！

记住这个比喻：

🎬 导演指示 + 🙋 观众提问 = 🤖 演员表演
